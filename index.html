Instagram Border / Canvas Calculator

Inputs are already scaled image dimensions (px). No upscaling is done here.
Portrait → 4:5 Square → 1:1 Landscape → 3:2 +10px on long edge

Input

Guideline implemented:
  1. Image size is fixed (you enter scaled values).
  2. Choose target aspect ratio by orientation.
  3. Expand canvas on the short edge only until the ratio is reached.
  4. With aspect lock, expand canvas by +10px on the image’s long edge.
  5. Center the image → borders split evenly.
